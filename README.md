
# 🚀 Arfadev.com – Modular Sanity.io Website Starter

A modern, flexible, and scalable website starter built with **Next.js** and **Sanity.io**, designed using a **modular architecture**.

Perfect for agencies, personal websites, or anyone who wants to learn and build with Sanity in a clean, structured way.

---

## ✨ Overview

This project helps you build dynamic, content-driven websites using **Sanity.io** as a headless CMS and a **block-based (modular) system** for page building.

Instead of hardcoding layouts, you can create pages using reusable modules directly from the CMS.

---

## 🧱 Content Structure

This project includes two main schemas:

- **Page Schema (`page`)**
  Used to build dynamic pages using modular sections

- **Blog Schema (`blogs`)**
  Used for creating and managing blog content

---

## 🏗️ Modular Architecture

Pages are built using **modules (content blocks)** such as:

- Hero Section
- Text Content
- Image Sections
- Call To Action
- وغيرها...

You can stack and reorder these modules directly in Sanity Studio to build fully dynamic layouts.

## 🚀 Getting Started

Getting started is simple:

### 1. Fork the repository

```bash
git clone https://github.com/umersaleem50/arfadev.com.git
cd arfadev.com
```

### 2. Setup environment variables

- Copy `.env.example` to `.env`
- Add your required **Sanity credentials**

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```
